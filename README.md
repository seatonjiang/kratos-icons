# Kratos Icons

Kratos 矢量图标库，包含了基础图标和品牌图标。

## 🚀 快速使用

在计划添加图标的页面 `<head>` 部分引入 `kratos-icons.min.css` 文件，请确保路径正确指向文件存放位置！

以下是一个示例 HTML 页面，其中在 `<head>` 部分包含文件链接，并在内容区域使用了图标，可以使用 `<i>` 或 `<span>` 标签实现：

```html
<head>
    <link href="/css/kratos-icons.min.css" rel="stylesheet" />
</head>t
<body>
    <i class="ks ks-kratos"></i>
    <span class="ks ks-kratos"></span>
</body>
```

## 📂 目录结构

```bash
kratos-icons
├── css
│   ├── kratos-icons.css
│   └── kratos-icons.min.css
├── demo
│   └── index.html
├── svgs
│   ├── admin
│   ├── brands
│   └── user
└── webfonts
    ├── kratos-icons.ttf
    ├── kratos-icons.woff
    └── kratos-icons.woff2
```
